# Chinese Name Spirit

> Discover Your Authentic Chinese Name - A personalized Chinese name generator for foreigners

## Project Overview

This project helps foreigners discover meaningful Chinese names based on their personality, preferences, and cultural background. Through an interactive quiz, users receive a personalized Chinese name with detailed cultural significance, meaning analysis, and a beautiful digital certificate.

## Features

- **Personalized Name Generation**: 5-question personality quiz to match you with the perfect Chinese name
- **Detailed Name Report**: Complete analysis including meaning, cultural significance, personality match, and lucky elements
- **Digital Certificate**: Beautiful PNG certificate with your Chinese name in calligraphy styles
- **Bilingual Reports**: Available in both English and Chinese
- **Blog Section**: Articles about Chinese naming culture, traditions, and meanings
- **SEO Optimized**: Structured data, meta tags, and sitemap for better search engine visibility

## How to Use

1. Clone or download this repository to your local machine
2. Open `index.html` in any modern browser (Chrome, Edge, Safari)
3. Answer 5 simple questions about your personality and preferences
4. Receive your personalized Chinese name with detailed report
